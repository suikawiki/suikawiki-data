*ABSTRACT
We develop distributed algorithms for self-organizing sensor
networks that respond to directing a target through a re-
gion. The sensor network models the danger levels sensed
across its area and has the ability to adapt to changes. It
represents the dangerous areas as obstacles. A protocol that
combines the articial potential eld of the sensors with the
goal location for the moving object guides the object incre-
mentally across the network to the goal, while maintaining
the safest distance to the danger areas. We give the analysis
to the protocol and report on hardware experiments using
a physical sensor network consisting of Mote sensors.
