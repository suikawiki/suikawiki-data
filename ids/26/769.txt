* Data model

[9] An [DFN[object]] has 
[DFN[[F[name]]]],
[DFN[[F[description]]]],
[DFN[[F[author]]]], and
[DFN[[F[license]]]],
which are initially [[null]].

[80] An [DFN[object]] has [DFN[[F[links]]]], which is initially an empty list.

[23] A [DFN[data set]] is an [[object]].

[24] A [[data set]] has
[DFN[[F[generator]]]],
which is initially [[null]].

[19] A [DFN[data set]] has
[DFN[[F[waypoints]]]],
[DFN[[F[routes]]]], and
[DFN[[F[tracks]]]], which are initially empty lists.

[25] A [DFN[point]] is an [[object]].

[26] A [[point]] has
[DFN[[F[latitude]]]], and
[DFN[[F[longitude]]]],
which are initially [[null]].

[43] A [DFN[person]] has
[DFN[[F[name]]]] and
[DFN[[F[email]]]],
which are initially [[null]].

[79] A [DFN[person]] has [DFN[[F[links]]]], which is initially an empty list.

[51] A [DFN[license]] has
[DFN[[F[holder]]]],
[DFN[[F[year]]]], and
[DFN[[F[page URL]]]],
which are initially null.
