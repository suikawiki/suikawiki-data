[1] [[リクルートポイント]]との統合の時緊急メンテみたいなのでgdgdしてたけどだいぶやばいシステムになってそう。

[2] リクルートのアカウントに対応する Ponta 側のアカウントが自動で作られたけど、
正式な Ponta の ID が発行されたというわけではないらしい。
手動で申し込み操作すると会員カードが送られてくる。
リクルートポイントと Pontaポイントも別になっていて、手動で移行するとリクルートポイントから
Ponta ポイントに切り替わる。(この移行は Ponta の ID 発行とは別。)
でも移行前でも Ponta 側のポイントは溜まっている。 

[3] リクルートのアカウントに自動で Ponta の ID が割り当てられた状態を解除して、
他の方法で作った Ponta 会員 ID に紐付ける方法は、なさそうに見える。

[4] リクルートから自動作成された Ponta 会員の会員 ID は、会員カードが送られるのを待たないとわからないのかな。
Ponta の Web サイトには上位数桁だけ見えていて、下位は伏せられている。
詳細を表示しようとすると、 Ponta の古い Web サイトに飛ばされるけど、
旧パスワードとかいうよくわからないものを聞かれる。その画面で会員 ID の下3桁だけ見ることができる。
(なので上位と下位はわかるけど、中間はわからない。)
旧パスワードなど元々存在しないので、新しく設定しようとすると、
会員IDを入れろと言われて詰む。
