* 文字コード

[4] 
[[中文]]入力用に、
[DFN[omron_udc_zh]]
と呼ばれる独自の[[符号化図形文字集合]]
([[94集合]]) を [[EUC]] に追加して使っていました。
[SRC[>>5]]

[3] 
[[cWnn]] は [[EUC-CN]] の [[CS0]] に追加していました。
[SEE[ [[EUC-CN]] ]]

[6] 
[[tWnn]] は [[EUC-TW]] の変種を使っていました。
[SEE[ [[euc-twnn]] ]]

[7] 
[[Mule]] が
[DFN[chinese-sisheng]]
と呼んで使っているものは、同じものを指しているようです。
[SEE[ [[Mule内部コード]], [[Fp]] ]]
[DFN[SiSheng特殊フォント]]とも呼ばれています [SRC[>>11]]。

[9] 
[[ctext]] の [[extended segment]] で使う[[文字コード]]名として、
次のものが登録されています。 [SRC[>>8]]
[SEE[ [[ctext]], [[extended segment]] ]]

,[CODE[omron_UDC]],omron User Defined Charset
,[CODE[omron_UDC_ja]],omron User Defined Charset for Japanese
,[CODE[omron_UDC_zh]],omron User Defined Charset for Chinese(Main land)
,[CODE[omron_UDC_tw]],omron User Defined Charset for Chinese(Taiwan)

[10] これらは [[Wnn]] で使われているものを指すと推測されますが、
詳細不明です。

[12]
