* とんでも解説本・雑誌記事によると HTML 3.x にあるらしい機能

- [[フレーム]] (HTML 3.0, HTML 3.2)
-- フレームが HTML 3.2 というデマを流したのは日本の M$。
<http://groups.google.com/groups?selm=E0vFuH.DK4%40vega.aichi-u.ac.jp>
-- HTML 3.2 の[[文書型宣言]]で堂々と [CODE(HTMLe)[[[frameset]]]] を使っている例文を載せている激しい解説書もある。
- [[スタイル・シート]]
-- 予約されているが、まだ使えない状態。
-- <http://www.microsoft.com/japan/msdn/library/ja/jpdnie40/htm/WebView.asp?frame=true>
によると、 [[CSS]] は HTML 3.2 の一部らしい。 
-- [CODE(HTMLe)[[[style]]]] tag は待ちに待った HTML 3.2
の目玉機能だと力説している解説書まである。
(その本には HTML 3.2 仕様書からの引用文があるが、
著者は本当に仕様書を読んだのか。)
- [CODE(HTMLe)[[[embed]]]] 要素型: [[NN]] の独自拡張だが、
IETF や W3C で検討中ではあった。
- [[ActiveX Control]] ([CODE(HTMLe)[[[object]]]])
-- HTML 3.2 にはまだない。
-- [[WinIE]] の独自拡張だが、 IETF や W3C で検討中ではあった。
- [CODE(HTMLe)[[[blink]]]] 要素型: [[NN]] の独自拡張
- [CODE(HTMLe)[[[nobr]]]] 要素型: NN の独自拡張
- [CODE(HTMLe)[[[multicol]]]] 要素型: NN の独自拡張
- [CODE(HTMLe)[[[multicolum]]]] 要素型: 解説本著者の妄想
- [[スクリプト]]
-- HTML 3.2 では予約されているが、定義がない状態。
- [CODE(HTMLe)[[[meta]]]] 要素型
-- [CODE(HTTP)[[[Refresh]]]] の利用など
