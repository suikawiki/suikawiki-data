1999年2月、 Eolas は Microsoft が Internet Explorer で米国特許 5,838,906
号を侵害していると提訴した。この特許は、 University of California が
1994年に出願し、1998年11月に認められたもので、同大学の研究者が設立した
Eolas に独占的にライセンスされている。その内容はハイパーテキスト文書内の記述
に基づいてプログラムを実行して処理させるシステムに関するものである。
HTML の仕様的にそのような実装方法が求められているわけではないが、
Netscape なども同様な方法で実現している。ただし、 Eolas
が提訴したのは現在に至るまで Microsoft だけである。

2003年8月、陪審員は Microsoft が特許を侵害しているとして
5億ドルの支払いを命じた。なお、後に、判事が陪審員に対して先行技術
の可能性がある Viola (Web ブラウザ, 1994) の存在を示さなかった
ことが問題とされている。また、陪審員が大企業の Microsoft
に対してはじめから良い印象を持っていない可能性があることや、
技術的な詳細を理解していたかにも疑問が持たれている。

2003年9月から10月にかけて、 Microsoft は Internet Explorer
の仕様の変更を決定し、この機能を利用した製品を出荷している
Macromedia, Adobe などはそれに対応した HTML 文書内の記述の
変更方法を発表した。この変更は既存の HTML 文書との深刻な
互換性の問題を引き起こすことに加え、元々の宣言的な記述を
手続き的な記述であるかのように変えてみたり、 Web
ブラウザ利用者の手間を増やしてみたり、面倒が増える割に
本当にそれで特許が回避できているのか怪しい。 Web
サイトの広告の手法にも一部影響があり、商業的な影響も計り知れない。

W3C はこの特許の影響を重大視して 2003年9月に HTML PAG (Patent Advisory Group)
を設立し、10月には特許の再審査を要請。先行技術として Viola や初期の
HTML 仕様案を提示した。これを受けて11月には特許商標庁が再審査を決めた。

2004年1月には判事が先の陪審評決を支持したが、 Microsoft は逆に
Internet Explorer の仕様変更を一旦保留することに決定。
これによって大きな互換性の問題は一時回避された。

2004年3月、特許商標庁は W3C の主張を取り上げ、特許は無効の可能性がある
との予備裁定を下した。5月、それに対して大学側が反論書を提出。
6月に Microsoft は1月の判決を不服として控訴した。

2004年8月、特許商標庁は特許を無効とする方向を表明した。

2005年3月、高等裁判所は一審を無効として差し戻す判決を下した。
しかし同月、 Microsoft は、その判決のうち賠償金の計算方法
に関する部分の再審を求めた。 Microsoft はマスター・ディスクを米国外
で製造しており、米国の特許法は適用されないと主張していた。

2005年9月、特許商標庁は特許が有効であると裁定した。
一度無効の予備裁定が出た後反論を受けてそれが覆るのはよくあることらしい。

2005年11月、最高裁判所は賠償金の計算に関する Microsoft の上告を棄却した。

現段階で特許商標局の再審査と賠償金に関する裁判は Eolas
の主張が認められているが、特許についての差し戻し審の判断は出ていない。
Viola の開発者は以前 Viola を特許の発明者に見せたと主張しており、
特許出願時に先行技術の情報を伝えなかった不正行為の疑いが持たれている。

もし Eolas の主張が認められた場合、 Internet Explorer
および同様の技術を使う他のブラウザや、それを使う製品や Web
サイトは修正を迫られる。 W3C などが開発している次世代の技術も
修正しなければならない部分が出てくるだろう。
そうなると影響の範囲は1999年よりも2003年よりも格段に大きい。

ところで、 Web の掲示板に投稿されていた情報が正しいなら、
Eolas は社員が1人、特許が数件、投資家が100人の企業らしい。
