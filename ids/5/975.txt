

* RFC 1623 (= STD 50)

- 『Definitions of Managed Objects for the Ethernet-like Interface Types』
- Ethernet 的界面型の管理物件の定義

> This memo defines a portion of the Management Information Base (MIB)
> for use with network management protocols in the Internet community.
> In particular, it defines objects for managing ethernet-like objects.

このメモは、 Internet 世間においてネットワーク管理プロトコルで使う
管理情報基盤 (MIB) の一部分を定義します。特に、 ethernet 的物体の
管理のための物体を定義します。

> This memo also includes a MIB module.  This MIB module corrects minor
> errors in the earlier version of this MIB: RFC 1398 [15].

このメモは MIB モジュールも含みます。この MIB モジュールは、
この MIB の以前の版 RFC 1398 の小さな誤りを訂正します。
