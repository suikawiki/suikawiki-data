[1] 
[DFN[[CODE[455]]]]
(この状態では method が妥当ではありません)
は、
[[RTSP]]
の[[状態符号]]の1つです。

* 構文

[4] 
[CODE[Allow:]] を使う[SHOULD[べき]]とされます。
[SRC[>>3]]

* 関連

[2] 
[[HTTP]]
には
[CODE[405]]
や
[CODE[501]]
があります。

* 歴史

[FIG(quote)[ [3] [[RFC 2326]] (RTSP) 11.3.6 455 Method Not Valid in This State

> The client or server cannot process this request in its current
state.  The response SHOULD contain an Allow header to make error
recovery easier.

クライアントあるいはサーバーは、
この要求を現在の状態では処理できません。
応答は誤り回復を容易にするために [CODE(RTSP)[[[Allow]]]]
頭を含める'''べき'''です。
]FIG]
