[[HTML Layout]] を [[WYSIWYG]] で
([[VB]] の窓編集画面のように) 編集したり、
それを [[HTML]] に埋め込んだり出来る。

[2] 当時はこれだ! と思ったのですが、生憎出力されるのは素の HTML
ではなく、 HTML Layout なる独自技術 (しかも HTML
に [CODE(HTMLe)[[[object]]]] として埋め込むことと、 [[SGML]]
っぽい構文であること以外 HTML と全然関係ない。)
だったので残念な思いをしました。1996年くらいでしたかな。

それからしばらくして、 HTML として同じような機能を実現するソフトウェアをいろいろなところが出してきました。
[[どこでも配置モード]]とか。

ところがそのようなのの出力は [CODE(HTMLe)[[[div]]]] 
厨で、とても使い物になりそうもありません。

interface は (というか大抵の draw 系ソフトウェアの interface
はそれと同じか。) ウェブには向かないというか、
余程がんばって作りこまないとまともな出力が得られなそうです。
(Windows という一つの比較的差が小さい環境でさえ、
font の設定で文字がはみ出して隠れたりみたいな問題が未だにたまにありますし。)
