* Internet Ninja (1)

[5] [[HTML]], [[画像]]
([[GIF]], [[JPEG]], [[BMP]]),
動画・音声 ([[AVI]], [[WAVE]],
[[MIDI]], [[QuickTime]]
mopvie),
その他データ一般を「ブック」
の要素として管理する。

画像は縮小表示される。
代わりに他の絵や文を指定することも出来る。

源 [[URI]] や注釈を保存できる。

[[WWWブラウザ]]や [[explorer]]
から [[D&D]] でデータを登録できる。

ブラウザのキャッシュからも登録できる。

登録データは再生したり、
[[Word]] などに貼り付けたり出来る。

[[#comment]]

* Internet Ninja 2

[6] 1997年12月5日。9800円。
[[NN2]]+, [[WinIE3.0]]+ 対応。
[[アスキーサムシンググッド]]。

[7] 「ブック」は入れ子 (木構造)
にできるようになった。

[CODE[コントロールバー]]という表示形態にすることで、
邪魔にならずに WWW
ブラウザを利用できる。
D&D や[CODE[現在のページ取込]]ボタンを押すとデータが取り込まれる。

保存済みデータを[[スライドショー]]表示できる。

指定した URI を、埋め込み画像やリンク先頁も含めて自動的に取り込むことが出来る。

時刻と URI を指定して自動的に取り込む
([[Web巡回]]) 機能が追加された。

ブラウザのキャッシュからデータを取り出すことも可能。

[8] [CODE[Ninja プロキシ]]機能で、
[[script]] や [[plugin]]
のデータなどを完全に取り込むことも可能。

[9] [CODE[Ninja クイックサーバ]]機能で、
保存データをサーバーとして提供可能。
[[イントラネット]]での利用を想定しているらしい。

[[#comment]]

* Internet Ninja 3

[12] Web ページのリンク関係を木表示可能に。

[[Y!]] や [[Infoseek]]
や[[千里眼]]など23種類の検索円陣から一つ選んで、検索結果を一気に得ることが可能。

[[Win32]] 対応、 9800円。
1998年。

[[#comment]]

* Internet Ninja 4

[2] 1999年4月16日発売。
Win32; WinIE3+, NN2+ 対応。

[3] 主な新機能:
-データ取り寄せの途中から再開
-[[検索円陣]]で検索してその結果の Web site
群をまとめて取り寄せる
-[[掲示板]]の収集にも対応しているらしい
-[[MPEG]]/[[AVI]] 動画の任意のコマを取り出す

[13] レジューム: 作業中断後、その続きから再開。
(HTTP の [CODE(HTTP)[[[Range]]]] つき要求)

取り寄せた Web 頁から画像のみを取り出して保管することも可能。

[[Susie]] plugin に対応。

[[#comment]]

「高速な[[シリアル]]・ドライバなどの導入で
