[1] [DFN[URP]] ([DFN[Uniform Resource Primitive]])
は [[Patrick Stickler]] が提案していた [[URI]]
の一分類 [WEAK[([[URL]] や [[URN]] と同じようなレベルの URI の種類)]]
です [SRC[PSTICKLER 2002]]。多くの [[URI scheme]] は伝統的に[[取出し]]が可能であることを想定していますが、
取出すまでもなく URI そのものが意味のある存在であるものが URP です。

[2] URP は、更に[[語彙]]としての URI である [[URT]]
(Uniform Resource Term) と値としての URI
である [[URV]] (Uniform Resource Value) に分けられます。
前者には[[言語札]]のようなものの URI 表現が、
後者には [CODE(URI)[[[data]]:]] URI のようなものが該当します。
