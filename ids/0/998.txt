* Web における Java

[8] [[Java]] は、はじめ [[Java applet]] として [[Webブラウザー]]に実装され、
広く普及しました。

[9] その後[[クライアント側]]の言語としては衰退しましたが、[[Webアプリケーション]]の[[サーバー側]]の言語としてよく用いられています。

[10] [[W3C DOM]] は [[Java]] と [[JavaScript]] を標準の言語として採用していました。
[[Webブラウザー]]業界の影響力が弱まった開発末期には、 [[W3C DOM]]
は [[Java]] を強く意識した [[API]] となっていました。
