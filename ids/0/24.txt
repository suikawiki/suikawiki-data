* はじめての SuikaWiki

[1] [DFN[[RUBY[[[SuikaWiki]]] [すいかゐき]]]] <http://suika.suikawiki.org/~wakaba/wiki/sw/n/> は、さまざまな話題を扱う 
[RUBY[[[wiki]]] [ゐき]] です。
どなたでも、どんな話題でも、新しい記事を書いたり、
既存の記事を読んだり修正したりできます。
[WEAK[(法律等の問題があるものを除く。)]]

SuikaWiki [[WikiEngine]] は、 [[WikiDatabase]] 内の情報を有機的に結び付けます。
閲覧者のみなさんは、情報から情報へと、[[ハイパーリンク]]を通して wiki
内を自由に探索することができます。

[2] 次の入力欄に、何か興味がある言葉を入れて、
[Q[[[#form:'%res (name => {WikiForm:Search});':'':'%output (form => 0);']]]]
してみてください。
[[#form:'%wfl--link-to-wikipage (content => {%text (label => {%xe--img (uri => {http://suika.fam.cx/~wakaba/archive/2005/sw-2});%res (name => {WikiForm:Search:Name});}p, id => mypage, size => 10);%wfl--hidden-mode (mode => {default}); %submit (label => {%res (name => {WikiForm:Search});}p);});':'':'%output(form=>0);']]
(検索後にこの頁に戻ってくるときは、お使いの [[WWWブラウザ]]の
[Q[戻る]]ボタンなどをお使い下さい。

おさがしの言葉に完全に一致する [[WikiPage]] があれば、それがでてきたはずです。
見つからなかった場合でも (見つかった場合でも)、
関係のありそうな WikiPage のリストが載っているはずです。
(もしどちらもなければ、現在この wiki にはおさがしの言葉に直接関係する情報はありません。
ただし、少し検索語を工夫してみれば、何か見つかるかもしれません。)

ほとんどすべての WikiPage には、何らかのリンクがあります。
そのリンクをたどると、さらに関連する別の情報を探すことができます。
(もしかするとこの wiki の外の情報にたどりつくかもしれません。
お使いのブラウザによっては、 wiki 内へのリンクと wiki 外のリンクが違って表示されます。)

[3] もし自分の知っていることについての情報がなければ、
ぜひともそれを書込んでみてください。ほとんどの WikiPage 
には、章の最後や頁の最後に記入欄が用意されています。
そこに書いて、送信してください。 (名前や電子メイルの欄もあるかもしれませんが、
必須ではありません。名前を省略すれば[[名無しさん]]になります。)

もし wiki になれていれば、入力欄から追記するだけではなく、
全文編集もお使い下さい。それぞれの WikiPage には[Q[編集]]モードへのリンクが用意されています。

記入の際は、普段は特に書式を意識する必要はありません。
しかし、文章の一部を強調したいとか、リストを書きたいとかいったような場合には、
[[Wikiの構文]]をご覧下さい。

[4] 注意: 誰でも自由に利用できる wiki の性質を維持するために、ライセンスに関する規定を設けています。
詳しくは [[Wiki//Page//License]] をご一読下さい。
