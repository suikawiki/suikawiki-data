[1] [DFN@en[Web 5.0]] は、 [[HTML 5]]
や [[HTML 5]] と同様の設計思想に基づき策定された仕様群により構築された
[[Web]] (の技術、あるいは [[Web]] そのもの) 
を指す[[冗句]]です。 [[Web 2.0]] をもじったもので、
主として [[WHATWG]] に賛同する人が冗談で使っている言葉でした。

[2] [[Web 5.0]] を支える技術には [[HTML 5]] と [[XHTML 5]]
の他に [[XML 5]]、[[HTTP 5]]、[[SVG 5]]、
[[CSS 5]] などがあります。
([[MathML 5]] もあったかも。)
[[HTML 5]] と [[XHTML 5]] 以外は現在のところ''架空''の仕様ですが、
今後ともこれらの仕様の[Q[問題]]が改善される見込みがなければ、
[[WHATWG]] の中の人達は本気でこれらの[Q[第5版]]を作り始めるかもしれませんでした。

[16] この言葉は2010年前後に使われていましたが、その後 [[HTML5]] は [[HTML Standard]]
となり、 [[DOM5]] (仮称) は [[Web DOM Core]] を経て [[DOM Standard]]
となるなど、「5」シリーズの時代が終わり、死語となりました。

* 関連仕様の一覧

[17] 実際に「5」がついていたもの:
[FIG(list)[
- [[HTML5]] → 現 [[HTML Standard]]
-- [[HTML5]] ([[HTML5]] 仕様書で規定された [[HTML文書]]) → 現 [[HTML]]
-- [[XHTML5]] ([[HTML5]] 仕様書で規定された [[XHTML文書]]) → 現 [[XHTML]]
-- [[DOM5 HTML]] ([[HTML5]] 仕様書で規定された [[DOM]]) → 現 [[DOM HTML]]
- [[XML5]]
- [[xml-stylesheet5]] → [[xml-stylesheet 1.1]] → 現 [[xml-stylesheet 1.0 2e]]
]FIG]

[18] 通称または概念として「5」がついていたもの:
[FIG(list)[
- [[DOM5]] → [[Web DOM Core]] / [[Web DOM Range]] → [[DOM4]] → 現 [[DOM Standard]]
- [[URL5]] → 現 [[URL Standard]]
- [[HTTP5]] → [[Tolerant HTTP Parsing]] が提案されたものの頓挫
]FIG]
