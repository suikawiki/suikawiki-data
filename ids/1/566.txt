[160] [DFN[[[W3C]] [[HTML Working Group]]]] は、 [[HTML5]] の開発を担当する [[W3C]]
の[[作業部会]]です。 [[HTML WG]] は2007年3月に発足しました。 [[W3C]] [[HTML Activity]]
に属しています。

;; なお、2007年3月以前の[[旧HTML WG]] は [[XHTML2 WG]] になりました。以前の [[HTML WG]]
に関しては[[旧HTML WG]] の項を、以後の[[旧HTML WG]] に関しては [[XHTML2 WG]]
の項をご覧ください。

[2] 2010年までは [[WHATWG]] と共同で [[HTML5]] の開発を行っていましたが、
開発方針の違いにより決別し、以後は独自仕様を出版しています。ただしその後も
[[WHATWG]] [[HTML Standard]] の変更を取り込んでいるようです (仕様書のバリエーションと機能により、
[[W3C]] [[HTML WG]] が数日から数年の範囲で遅れています)。 (ただし [[WHATWG]]
の [[Ian Hickson]] はこれを[[剽窃]]行為であると非難しています。)

[45] 2015年10月に [[Web Platform WG]] に改組されました。

* 仕様

[1] [[HTML WG]] は次の仕様を開発しています。
[FIG(short list)[
- [[HTML 5.0]]
- [[HTML 5.1]]
- [[HTML5 Differences from HTML4]]
- [CODE(HTMLa)@en[[[longdesc]]]]
- [CODE(HTMLe)@en[[[main]]]]
- [CODE(HTMLe)@en[[[picture]]]]
- [[HTML 5: The Markup Language]]
- [[HTML5: Techniques for providing useful text alternatives]]
- [[Polyglot Markup]]
- [[Public Identifiers for entity resolution in XHTML]]
- [[HTML Microdata]]
- [[DOM4]]
]FIG]

[3] [[HTML]]、[[Microdata]]、[[DOM]] は [[WHATWG]] の仕様書を元に独自の変更を加えたものです。
それ以外は独自に開発されている仕様書です (その多くは [[WHATWG]] で技術的理由から却下されたものです)。

* WHATWG との関係

[161] [[HTML5]] は元々 [[WHATWG]] で「[[Web Applications 1.0]]」として開発されてきました。
[[W3C]] [[HTML WG]] は発足直後の投票で、「[[Web Applications 1.0]]」を「[[HTML5]]」
として採用して開発を進めること、 [[WHATWG]] の中心人物である [[Ian Hickson]] ([[Google]])
を[[編集者]]とすることなどを決めました。

[162] それ以後 [[HTML WG]] は [[WHATWG]] と共同で [[HTML5]]
の標準化作業を行っています。 [[HTML5]] 仕様書案は両部会から出版されていますが、
目次以降はすべて同じ内容となっています。

[163] 議論の傾向として、 [[HTML WG]] では政治的話題が、 [[WHATWG]] では技術的話題が多い感じです。

* Forms Task Force

[164] [[HTML WG]] と [[Forms WG]] との共同で [[Forms Task Force]]
が設立されましたが、何の成果も出さないまま事実上解散しています。
詳しくは [[Forms Task Force]] の項をご覧ください。
