#?SuikaWiki/0.9

- [1] XML Namespace Partition。[[XML名前空間]]では、[[要素型名]]および[[属性名]]は合わせて3つの[[区画]]に分類されます。この「区画」は、従来的な意味 (非 XML 名前空間的意味) の「名前空間」に相当するものです。
- [2] ''Namespaces in XML : A.2 XML Namespace Partitions'' <http://www.w3.org/TR/REC-xml-names/#ns-breakdown>
- [3] [[全要素型区画]] : 全ての要素型名が所属する区画。
- [4] [[大域属性区画]] : 大域属性 ([[名前空間名]]で名前が修飾された属性) が所属する区画。
- [5] [[要素型毎区画]] : 各要素型毎に存在する、名前空間の無い属性の所属する区画です。
