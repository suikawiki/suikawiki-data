> User agents MUST make information about the state of TLS protection available. The '''['''Definition: TLS indicator''']''' SHOULD be part of primary user interface during usage modes which entail the presence of signaling to the user beyond only presenting page content. Otherwise, it MUST be available through secondary user interface.
