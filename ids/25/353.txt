

* 開発

開発は色々な方法がありますが、デスクトップ版の Chrome を使って基本的な動作を確認してから、他のブラウザーでも確認するのがおすすめです。ブラウザーの窓の幅を狭めれば、タブレットやスマートフォンでの表示を想定した確認もすぐに行なえます。

;; Chrome の開発者ツールを使えばよりタブレットやスマートフォンに近づけた動作検証もできますが、細かな調整以外ではそこまでしなくても良いでしょう。

Chrome で確認しておけば、 Opera はもちろん、 Firefox でもそのままで意図通りに表示・動作することがほとんどです。 

IE や Edge は他のブラウザーよりも何世代か古い技術にしか対応していないことが多いので、そのままでは意図通りにならないことがよくあります。予め工数に算入しておきましょう。常に IE で確認するのでは手間がかかりすぎるので、開発の末期にまとめて修正するのがよいかもしれません。 (それでは手戻りが多くなりすぎるおそれがあるので、開発初期の方針がおおむね定まった段階で一度動作確認しておいた方がいいかもしれません。)

;; Windows 以外で開発するときや他のバージョンを試したい時は、
Microsoft が無料で配布している IE や Edge の入った VM を使いましょう。
