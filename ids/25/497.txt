> You can create a drop cap—including dropped, raised, or sunken initial letter effects—using Safari 9.0's support of the initial-letter CSS property:
> initial-letter: 3 3;
> The first number defines number of text lines the letter will span. The second number defines the number of lines the letter should sink.
