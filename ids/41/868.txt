* 比較

[5] 
[[ISO/IEC 8859-2]] は [[ISO/IEC 8859-1]] と似ていますが、
いくらか出入りがあって互換性はありません。

[8] 
[[ISO/IEC 8859-1]] と [[Windows-1252]] の [CC[13/0]] は [CC[U+00D0]] ([CH[Ð]])
ですが、
[[ISO/IEC 8859-2]] と [[Windows-1250]] の [CC[13/0]] は [CC[U+0110]] ([CH[Đ]])
です。



[6] 
[CITE[Encoding Standard]] の [[ISO-8859-2]]、
[[Unicode Consortium]] の [[ISO/IEC 8859-2]] の変換表、
[[ICU]] の [DFN[[CODE[ibm-912_P100-1995]]]]
は1対1対応関係についていえば同等です。

[7] 
[[Windows-1250]] は [[ISO/IEC 8859-2]]
とよく似ていますが、いくらか出入りがあって互換性はありません。

[9] 
[[Windows-1250]] は [[Windows-1252]] ([[ISO/IEC 8859-1]] の拡張)
とよく似ていますが、いくらか出入りがあって互換性はありません。

[10] 
[[ICU]] の [DFN[[CODE[ibm-5346_P100-1998]]]] は
[[Windows-1250]]
と1対1対応関係についていえば同等です。

[11] 
[[ICU]] の [DFN[[CODE[ibm-1250_P100-1995]]]]
は [[Windows-1250]] の旧版を表し、[[ユーロ記号]]を除けば同等です。
