

* 快適 (かもしれない) Wineoze ネットワークの活用☆

[1] わかばの自室の計算機は最近1台増えて (お下がりね(笑))
3台になったんですが、画面がないのやらあってもブラウン管
だし見てるのが辛いのなんのって, その上ちょっとの距離だけど
移動が面倒だし:-p

で、どうしたかと言いますと、全部 Windoze なもんですから、
常用でない2台を VNC server にしてしまって、今使ってる
FIVA (WinMe) が client になって覗いてます。

で、それだけだと遅くていらいらするので (ネットワークが 10base
だから遅いのか、そもそもそいつらの MPU や memory のせいなのか...)
そいつらはハードディスクをすべて共有にしてしまって、
ファイル操作とかはネットワーク越しにこっちでやってしまいます。
(てことはやっぱり遅いのは、画面描画の送受信と再現なんだろうか。)

そこまですると一体何のために使っているのかよくわかりませんが...
とりあえず、 Win95 OSR2 や Win98 が入っているので、
環境の差異による比較実験に使えることがわかりますた。
(とは言っても perl や cygwin の中だけでほとんど生活してると、
その程度の環境の差異なんてないようなものですがね...)
