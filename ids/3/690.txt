
[1] 【[[XML]]】 XML [[型録]]では、次の種類の[DFN[型録項目]]
(catalog entry) が定義されています。
,[[catalog]]
,[[delegatePublic]]
,[[delegateSystem]]
,[[delegateURI]]
,[[group]]
,[[nextCatalog]]
,[[public]]
,[[rewriteSystem]]
,[[rewriteURI]]
,[[system]]
,[[uri]]

いずれも、 XML 的には[[名前空間名]]が
[CODE(URI)[[[urn:oasis:names:tc:entity:xmlns:xml:catalog]]]]
の[[要素型]]です。

[2] XML 型録仕様書附属書 E 
<http://www.oasis-open.org/committees/entity/spec.html#a.tr9401>
では、 [[TR9401]]
の XML 表現のために、次の追加の型録項目を定義しています。
いずれも XML 名前空間名は
[CODE(URI)[[[urn:oasis:names:tc:entity:xmlns:tr9401:catalog]]]]
です。
,[[doctype]]
,[[document]]
,[[dtddecl]]
,[[entity]]
,[[linktype]]
,[[notation]]
,[[sgmldecl]]

- [4] >>3 TR 9401 は [[SGML型録]]の仕様のことです。 SGML 型録にある機能のうち SGML にあって XML にはない機能に対応するものは XML 型録の規格本体には含まれませんでしたが、旧来の SGML 型録を XML 型録に変換する際に便利なように >>2 の要素型が参考として定義されているようです。
