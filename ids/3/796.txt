

* Bag 要素 (RDF)

[1] [[RDF]] の container 要素の1つで、資源又はリテラルの順序のない列挙に使います。
[[HTML]] の [[ul]] 要素に相当するでしょうか。

内容には0個以上の RDF:[[li]] 要素を持ちます。
(RDF:*/)[[ID]] 属性が使えます。

[2] 同様の要素で、順序のある列挙のための RDF:[[Seq]] 要素,
選択肢の列挙のための RDF:[[Alt]] 要素があります。

- [3] ''Resource Description Framework (RDF) Model and Syntax Specification: Containers'' <http://www.w3.org/TR/1999/REC-rdf-syntax#containers>
- [4] 出現できる場所は、 RDF:[[Description]] 要素が出現し得る場所。
