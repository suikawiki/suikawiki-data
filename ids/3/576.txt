[1] [DFN[NetObjects FUSION 3.0]]
(三菱商事、オープン価格、1998年)
は、 [[Webサイト]]管理に重点を置いた
[[WYSIWYG]] 型 [[AuthoringTool]] です。

[2] [DFN[サイト・ビュー]]で Web
サイトの木構造をつくり、
それぞれの構成 Web ページを[DFN[ページ・ビュー]]で編集します。

Page view は WYSIWYG ですが、
[[ワープロ]]型ではなく
[[DTP]] ソフトウェア型の object
配置をおこないます。

[[DHTML]] に対応(謎)。
メニューの選択だけで DHTML
を使えます(謎)。

[[CSS]], [[Shockwave]]
[[Director]]/[[Flash]],
[[QuickTime]] に対応(謎)。

対象ブラウザの設定機能: IE か NN
か、その版を選択可能。
(DHTML 出力に影響するらしい。)

[3] [DFN[NetObjects FUSION Personal 2.0]]: >>1 と同時発売。
