> Description: the application mode is for convenience web application and game developers set of instructions, opened by a meta tag, when entering the application mode, the browser will automatically adjust the following parameters:
> (parameter: state)
> Full screen: effective, by meta or JS API calls from the full screen
> Long press menu: failure, through the JS API call to take effect
> The default browser gesture: failure, through the JS API call to take effect
> Typesetting: standard mode, through the meta or JS API call other layout mode
> The picture shows: effective enforcement
> Night mode: failure, by meta or JS API called enable night mode
> Parameters:
> application: Application mode
> Usage:
> 1 <!--  Application mode  -->
> 2 <meta name="browsermode" content="application"/>
> Versioning support: 8.6+
