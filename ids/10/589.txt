*Oh my Windoze 95!

[1] 今朝[[ヒカルの碁]]を見ようとして (あとで書く今日の[[冬日]]参照。)
Woody をつけたんですが、テレビ起動までに5分かかりますた。
(いつものことではありますが。) 
従ってそれが普通のテレビであったなら見れたはずの5分が失われますた。
(普通のテレビでみればいいという説もある。)

というわけでさすがに驚きあきれて(謎)、 Windoze 98 FAT32
partition (なんと Office 2000 とか IE5 とか Netscape 6
が入ってた! (遅くて誰も使わないけど。)) を丸ごとでりって、
restore FD & CD で 1時間かけて Windoze 3.1 にしますた。
(それでも遅いのは気のせいです。)

さすがに Windoze 3.1 はアレゲなんで Windoze 95 にしたい
ところですが、 Windoze 95 の CD が見つかりません。
Windoze 95 OSR2.1 の CD ならあるけど、これ使ったら
折角 Windoze 3.1 入れたのになあ。 (PD/CD drive がもう
かなりやばくて、何回に1回しかちゃんと CD 読んでくれないし。)

もしこのまま Windoze 3.1 にするとしたら、 Adaptec SCSI
driver 入れて scanner が動くようにしないといけないし
(FD 入れ替えがめんどい!), ネットワークがつながらないと
これまた大変だし。 (Winsock16 をどっかで探してこれば
TCP/IP も使えると思うけど、 Windoze のファイル共有が
使えぬ...)

コマタ。 (2002-10-27)

[4] 結局 Windoze 3.1 を消して (deltree C:\WINDOWS),
アップグレード版でない Windoze 95 を入れることにしますた。

いつも (昔は年に数回 Windoze 95 の install してましたからねぇ。)
思うんですが、どうして中止ボタンを押しやすくしておくんでしょうか。

-0. マウスで押す
-1. Focused なので Enter 一発で押される状態
-2. Function key 3 (F3) が割り当ててある
-3. Access key F (= Alt + F) が割り当ててある

というように、非常に押しやすい状態であります。ですから、
もしや [[M$]] は install して欲しくないんでは、と思ってみたりも
したものです。
