> Netscape Object Signing is a protocol that lets developers create applications that can request fine-grained access to a user's system resources. The user then decides whether to grant the request.
> Object signing offers users the following basic features:
> Identification
> Capabilities-based Java
> Tamper detection
> Ability to sign any object
> Open-standards support
> Cross-platform flexibility
